## Как запустить локально

Убедитесь что у вас скачать python и pip

В папке проекта:

1. python -m venv .venv
2. .venv\Scripts\active (Windows), source .venv\bin\activate (Mac)
3. pip install -r r.txt
4. python manage.py migrate
5. python manage.py runserver

Сервер запуститься на localhost:8160
